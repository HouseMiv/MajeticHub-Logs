MajesticHUB Logs Stacker
======================

## Описание
MajeticHub-Staker Logs - это веб-инструмент, предназначенный для обработки и агрегации данных из столбца "Punish List". Он позволяет пользователям вводить данные о наказаниях (ID, тип наказания, время и имя) и автоматически обрабатывать их, суммируя повторяющиеся наказания для одного и того же ID и сортируя результаты в определенном порядке. Проект разработан с использованием HTML, JavaScript и CSS.

## Особенности
- Пользователи могут легко вводить данные и получать результаты без необходимости изучения сложных инструкций.
- Инструмент корректно отображается на компьютерах, планшетах и смартфонах, что делает его доступным для пользователей с любых устройств.

## Структура проекта
- `README.md` - файл с описанием проекта
- `favicon.ico` - иконка сайта

## Пример использования:
Предположим, у вас есть следующие данные в столбце "Punish List":
```
ID:123;PUNISH:ajail;TIME:300;NAME:1.3 ПГО (ИРП) by Miv;
ID:123;PUNISH:ajail;TIME:400;NAME:1.3 ПГО (ИРП) by Miv;
ID:456;PUNISH:ban;TIME:1000;NAME:1.3 ПГО (ИРП) by Miv;
ID:789;PUNISH:mute;TIME:600;NAME:1.3 ПГО (ИРП) by Verum;
ID:123;PUNISH:warn;NAME:1.3 ПГО (ИРП) by Verum;
ID:136548;PUNISH:ajail;TIME:40;NAME:1.3 ПГО (ИРП) by Verum;
```
После обработки эти данные будут преобразованы в следующую таблицу:
```
/ajail 123 700 1.3 ПГО (ИРП) by Miv, 1.3 ПГО (ИРП) by Miv
/ajail 136548 40 1.3 ПГО (ИРП) by Verum
/ban 456 1000 1.3 ПГО (ИРП) by Miv
/mute 789 600 1.3 ПГО (ИРП) by Verum
```

Этот инструмент помогает администраторам форума эффективно управлять и анализировать наказания, обеспечивая удобное представление информации и возможность быстрого принятия решений.
